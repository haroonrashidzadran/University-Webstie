### Colors
The website uses CSS variables for easy customization:
```css
:root {
    --primary-color: #1e3a8a;
    --secondary-color: #0ea5e9;
    --accent-color: #f59e0b;
    --dark-color: #1f2937;

    --border-radius-lg: 50px;

    --transition-speed-slow: 0.5s;

```
